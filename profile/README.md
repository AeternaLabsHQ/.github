## Aeterna Labs

AI tools with a focus on practical use..

Most open-source work lives under [syswave-dev](https://github.com/syswave-dev). This org holds the projects that outgrew the personal-tool stage.

### Projects

- **[PullMD](https://github.com/AeternaLabsHQ/pullmd)**: self-hosted service that converts almost anything to Markdown. Web pages, documents, images, audio, YouTube. Ships with a PWA, REST API, MCP server, and a Claude Code skill.
- **[Claude Code Stats](https://github.com/syswave-dev/claude-code-stats)**: dashboard for Claude Code sessions, built from the local JSONL logs.

### Approach

- Tools where the care is visible.
- Honest about what works and what doesn't.
- AI as an amplifier of your own thinking, not a replacement for it.
